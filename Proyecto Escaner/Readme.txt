*Usamos python3.10.0

*parar crear el .exe use el modulo "cx_Freeze"
	-el cual se instala con "pip install cx_Freeze"
	-el cual usa el archivo setup.py y la consiguracion que tiene hay para crear el .exe
	-para crear el .exe se hace "python setup.py build", el cual crea build y dentro de build estara el .exe

*instale todas las herramientas especificadas en el archivo "requerimientos.txt"

*las cuales se pueden instalar directamente con "pip install -r requerimientos.txt" o en su defecto "pip install -r --user requerimientos.txt"

*Las carpetas y los archivos multimedia deben manetener sus nombres, no importa si cambios por otros archivos, pero los nuevos archivos deben llamarse igual.
